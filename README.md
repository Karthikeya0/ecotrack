# EcoTrack 🌱

**EcoTrack** is an AI-powered carbon footprint tracker designed to help individuals log their daily habits, calculate their real-time CO₂ emissions, and receive personalized eco-friendly suggestions. 

## Features

- **Daily Logging**: Easily log daily habits across Travel, Food, and Energy.
- **Real-Time CO₂ Calculations**: Instantly see the environmental impact of your daily choices.
- **AI-Powered Suggestions**: Get personalized tips generated by AI on how to reduce your carbon footprint based on your specific logs.
- **Progress Tracking**: Maintain streaks and earn badges for consistent eco-friendly behavior.
- **Demo Mode**: Instant access to explore features without needing an account.
- **Offline Resilience**: Built with modern error boundaries and fallback mechanisms for a smooth user experience.

## Tech Stack

- **Frontend**: React (Vite), Tailwind CSS, Chart.js
- **Backend Services**: Firebase (Authentication & Cloud Firestore)
- **APIs Integration**: Carbon Interface API (Calculations), Groq AI (Suggestions)
- **Routing**: React Router DOM (with Lazy Loading for performance)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn
- Firebase Account (for database and auth)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Karthikeya0/ecotrack.git
   cd ecotrack
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   - Rename `.env.example` to `.env`
   - Fill in your Firebase and API configuration keys.

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
